Hangman Game Instructions

Requirements:
These two files must be kept in the same folder in order for the code to execute properly:
1. IlkerDuman_Python_04_01_Project_PythonGameDevelopment.ipynb
2. 4000_words


RULES:

Start:
Run the Hangman game script.
Enter your name when prompted.

Difficulty:
Choose 'easy', 'medium', or 'hard'.

Gameplay:
guess one letter at a time.
Avoid getting the hangman fully drawn.

Win or Lose:
Correct guesses prevent hangman drawing.
Win by guessing the word; lose if hangman is complete.

Play Again:
After each game, choose to play again or exit.

Enjoy:
Have fun playing Hangman!
